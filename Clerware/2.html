Clerware Admin / 科力锐 超级管理员

Posted on 2020-02-14 Edited on 2021-04-10 In Clerware Views: Changyan:
Symbols count in article: 1.1k Reading time ≈ 3 mins.

  本文是针对科力锐灾备一体机（clerware 智动全景灾备系统）超级管理员 admin 主要操作及功能的介绍，为博主的个人笔记，未必是正确操作，勿将其作为科力锐官方技术文档，仅供参考！

本文针对的是 超级管理员，如需 系统管理员 相关操作，请参考 《科力锐 系统管理员》

超级管理员 admin 操作

配置网络

  网络管理 → 网络设置 → 配置IP → 试用

配置完设备的 IP 后，点击的是“试用”，会有10分钟的试用期。通过设置的新 IP 进行登陆后，如果确认可以使用，需要到刚才配置设备 IP 的界面，点击“确认”从而生成永久配置。如果超过10分钟未点击“确认”，设备 IP 会回滚到原来的配置。

设置系统管理员

  用户管理 → 用户管理 → 新建 → 设置用户信息及功能授权 → 确定

新建用户初始密码是：123456

分配系统管理员存储配额

  存储管理 → 存储配额管理 → 内部存储单元 → 添加用户配额 → 选择用户进行相应配置 → 确定

超级管理员其余功能项

网络管理

网络设置

PXE 设置（不常用）

  目标机如果是裸机的情况下，另一种引导方式，以网卡的形式启动起来，就是将ISO用网络的方式推送到目标机，然后让它启动，一般用不到

路由配置

  给灾备系统添加路由，使其可以访问另一网络

路由检测（常用）

  用于测试灾备系统与目标主机能否通信

消息与通知设置

  用于设置灾备情况告警

用户管理

  用于配置、解锁系统管理员。系统管理员输错密码达一定次数被锁定，在此解锁

设置

系统更新

系统升级前需要关闭所有备份计划，确认无正在进行任务

客户端管理

  所有系统管理员的客户端在此都能查看

授权管理

  查询灾备一体机的序列号及型号，以及查看或上传灾备授权

系统设置

维护

  将密钥发送给科力锐技术支持，登陆后台 root 的密码就是：admin 密码+密钥解密后的明文

后台管理

  设置是否开启灾备一体机的 SSH 服务

密码策略（等保相关）

登陆设置（等保相关）

时间设置

  配置 NTP

VT（虚拟化平台相关）

  如虚拟化平台底层不支持，则不使用 VT

网络设置（虚拟化平台相关）

  如 MTU 值不是1500，可在此修改，以及设置 KVM 主机是否可通过 VNC 登陆
