The QRadar Ecosystem – Intelligent Detection / QRadar 生态系统——智能检测

  1. Predict and prioritize security weaknesses / 预测并优先处理安全弱点
    • Gather threat intelligence information / 收集威胁情报信息
    • Manage vulnerabilities and risks / 管理漏洞与风险
    • Augment vulnerability scan data with context for optimized prioritization / 使用“背景档案”扩充漏洞扫描数据以优化优先级
    • Manage device configurations (firewalls, switches, routers, IPS/IDS) / 管理设备配置（防火墙、交换机、路由器、入侵防御系统/入侵检测系统）
  2. Detect deviations to identify malicious activity / 检测偏差以识别恶意活动
    • Establish baseline behaviors / 建立基线行为
    • Monitor and investigate anomalies / 监测并调查异常情况
    • Monitor network flows / 监测网络流量
  3. React in real time to exploits / 对攻击的实时反应
    • Correlate logs, events, network flows, identities, assets, vulnerabilities, and configurations, and add context / 对日志、事件、网络流量、身份、资产、漏洞和配置进行关联，并添加“背景档案”
    • Use automated and cognitive solutions to make data actionable by existing staff / 使用自动化和认知解决方案，使现有工作人员可以对数据采取行动

What is Security Intelligence? / 什么是安全智能？

The real-time collection, normalization, and analytics of the data generated by users, applications, and infrastructure that impacts the IT security and risk posture of an enterprise / 实时收集、规范化并分析由用户、应用程序和基础设施产生的影响企业 IT 安全和风险状况的数据

IBM QRadar Vulnerability Manager / IBM QRadar 漏洞管理器

Scan, assess, and remediate vulnerabilities / 扫描、评估并修复漏洞

  • Contains an embedded, well proven, scalable, analyst recognized vulnerability detection engine that detects more than 70,000 vulnerabilities / 包含一个嵌入式的、经过充分验证的、可扩展的、分析师认可的漏洞检测引擎，可检测 70,000 多个漏洞
  • Integrates into the QRadar ecosystem / 集成至 QRadar 生态系统中
  • Is present on all QRadar event and flow collector and processor appliances (QRadar 7.2 and up) as well as QRadar data nodes (QRadar 7.2.8 and up) / 存在于所有 QRadar 事件和流量收集器和处理器设备（QRadar 7.2及以上版本）以及 QRadar 数据节点（QRadar 7.2.8及以上版本）
  • Integrates with endpoint management (IBM BigFix), web application security (IBM AppScan), database security (IBM Guardium), and network management (IBM Security SiteProtector) / 与端点管理（IBM BigFix）、网络应用安全（IBM AppScan）、数据库安全（IBM Guardium）和网络管理（IBM Security SiteProtector）集成
  • Leverages QRadar Risk Manager to report which vulnerabilities are blocked by your IPS and FW / 利用 QRadar 风险管理器来报告哪些漏洞被你的入侵防御系统和防火墙所阻挡
  • Uses QFlow report if a vulnerable application is active / 如果一个脆弱的应用程序处于活动状态，则使用 QFlow 报告
  • Presents a prioritized list of vulnerabilities you should deal with as soon as possible / 提出一个你应该尽快处理的漏洞的优先级列表

IBM QRadar Risk Manager / IBM QRadar 风险管理器

Scan, assess, and remediate risks / 扫描、评估并补救风险

  • Network topology model based on security device configurations enables visualization of actual and potential network traffic patterns / 基于安全设备配置的网络拓扑结构模型使实际和潜在的网络流量模式可视化
  • Policy engine correlates network topology, asset vulnerabilities and configuration, and actual network traffic to quantify and prioritize risk, enabling risk-prioritized remediation and compliance checking, alerting, and reporting / 政策引擎将网络拓扑结构、资产漏洞和配置以及实际网络流量联系起来，以量化和优先处理风险，实现风险优先的补救和合规性检查、警报和报告
  • Centralizes network security device configuration data and discovers configuration errors; monitors firewall rule activity / 集中网络安全设备配置数据并发现配置错误；监控防火墙规则活动
  • Models threat propagation and simulates network topology changes / 建立威胁传播模型，模拟网络拓扑结构变化

IBM QRadar SIEM / IBM QRadar 安全信息和事件管理

Web-based command console for Security Intelligence / 基于网络的安全智能命令控制台

  • Delivers actionable insight, focusing security teams on high-probability incidents / 提供可操作的洞察力，使安全团队专注于高概率事件
    Employs rules-based correlation of events, flows, assets, topologies, and vulnerabilities / 采用基于规则的事件、流量、资产、拓扑结构和漏洞的关联性
  • Detects and tracks malicious activity over extended time periods, helping uncover advanced threats often missed by other solutions / 在较长的时间段内检测和跟踪恶意活动，帮助发现其他解决方案经常忽略的高级威胁
    Consolidates “big data” security incidents within purpose-built, federated database repository / 将“大数据”安全事件整合到专门建立的联合数据存储库中
  • Provides anomaly detection to complement existing perimeter defenses / 提供异常检测以补充现有的边界防御措施
    Calculates identity and application baseline profiles to assess abnormal conditions / 计算身份和应用程序基线概况，以评估异常情况
  • Provides deep visibility into network, user, and application activity / 提供对网络、用户和应用程序活动的深度可视性
  • Provides reliable, tamper-proof log storage for forensic investigations and evidentiary use / 为取证调查和证据使用提供可靠、防篡改的日志存储

  3. Activity baselining and anomaly detection / 活动基线和异常情况检测
    1. User activity / 用户活动
    2. Database activity / 数据库活动
    3. Application activity / 应用程序活动

  4. Offense identification / 攻击行为识别
    1. Credibility / 可信度
    2. Severity / 严重性
    3. Relevance / 相关性

Providing functional context / 提供功能性的“背景档案”

To enable security analysts to perform investigations, QRadar SIEM correlates information such as: / 为了使安全分析人员能够进行调查，QRadar 安全信息和事件管理将以下信息关联起来，例如：

  • Point in time / 时间点
  • Offending users / 违规用户
  • Origins / 来源
  • Targets / 目标
  • Asset information / 资产信息
  • Vulnerabilities / 漏洞

  • Cognitive analytics / 认知分析法

Network flow analytics / 网络流量分析

  • Provides insight into raw network traffic / 提供对原始网络流量的洞察力
    Attackers can interfere with logging to erase their tracks, but they cannot cut off the network (flow data) / 攻击者可以干扰记录以消除他们的痕迹，但他们不能切断网络（流量数据）
  • Allows deep packet inspection for Layer 7 flow data / 允许对第七层流量数据进行深度数据包检查
    Pivoting, drill-down, and data-mining activities on flow sources allow for advanced detection and forensics / 对流量源进行透视、钻取和数据挖掘活动，可以进行高级检测和取证
  • Helps to detect anomalies that might otherwise be missed / 有助于检测可能被忽略的异常情况
  • Helps to detect zero-day attacks that have no signature / 有助于检测没有签名的零日攻击
  • Provides visibility into all attacker communications / 提供对所有攻击者通信的可见性
  • Uses passive monitoring to build asset profiles and classify hosts / 利用被动监测来建立资产档案和对主机进行分类
  • Improves network visibility and helps resolve traffic problems / 提高网络可视性，帮助解决流量问题

    3. Augment your analysts’ knowledge and insights with QRadar Advisor with Watson / 通过QRadar Advisor with Watson增强您的分析员的知识和洞察力
  2. Open Ecosystem / 开放的生态系统

    3. Open APIs allow for custom integrations and apps / 开放的API允许自定义集成和应用程序
  3. Deep Threat Intelligence and Analysis / 深度威胁情报和分析
    1. IBM X-Force Exchange helps you stay ahead of the latest threats and attacks / IBM X-Force Exchange帮助你领先于最新的威胁和攻击
